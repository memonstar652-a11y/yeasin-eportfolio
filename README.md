# Yeasin Arafat ePortfolio

A responsive personal ePortfolio built with HTML, CSS and JavaScript.

## Files

- `index.html`: Page structure and portfolio content
- `style.css`: Responsive layout and design
- `script.js`: Mobile navigation and scroll animations
- `assets/Yeasin_Arafat_CV_and_Cover_Letter.pdf`: Downloadable CV and cover letter

## Personalize the website

1. Add a profile photo to the `assets` folder and replace the initials card in `index.html`.
2. Upload a short introduction video to YouTube as unlisted and replace the video placeholder with a YouTube iframe.
3. Add real GitHub project links to the project cards.
4. Review all employment, education and contact information before publishing.

## Publish with GitHub Pages

1. Create a new GitHub repository and upload all files in this folder.
2. Open repository **Settings → Pages**. Under **Build and deployment**, select **Deploy from a branch**, choose `main` and `/(root)`, then save.
3. Wait for deployment. Your website will appear at:
   `https://YOUR-USERNAME.github.io/REPOSITORY-NAME/`
